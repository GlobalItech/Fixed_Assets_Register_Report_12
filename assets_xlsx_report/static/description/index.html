Fixed Assets Register Report

Fixed assets register report is very useful to get detailed information about a company assets.

The report can be generated easily from the report wizard in few clicks.

(1): To get the report, perform following: Accounting > Adviser > Asset Excel Report






(2): On the report wizard, select the period and then select the asset category which you want to get the report of.




(3): As shown in the picture below, the report shows the detailed purchase and depreciation related information of all the assets of the selected category.
